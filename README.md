# Kayra Cam Balkon 🌟
<img width="1920" height="4295" alt="FireShot Capture 043 - React App -  localhost" src="https://github.com/user-attachments/assets/b1cf314d-b563-48b1-b5a4-3c187705f202" />

Kurumsal tanıtım ve iletişim amaçlı hazırlanmış **React + Tailwind CSS** tabanlı tek sayfalık (One Page) web sitesi.  
Proje, mobil ve masaüstü cihazlar için **responsive** tasarlanmıştır.

---

## 🚀 Özellikler
- ⚡️ **React** ile component tabanlı yapı  
- 🎨 **Tailwind CSS** ile modern ve responsive tasarım  
- 🖼️ Fotoğraf galeri alanı  
- 🎥 Video destekli Hero bölümü  
- 📍 Google Maps entegrasyonlu iletişim sayfası  
- 📧 EmailJS ile iletişim formu (formdan mail gönderme)  
- 📱 Mobil uyumlu **hamburger menü** ve sağdan açılır navigasyon  
- 🔗 Footer’da sosyal medya butonları  

---

## 🛠️ Kullanılan Teknolojiler
- [React](https://react.dev/)  
- [Tailwind CSS](https://tailwindcss.com/)  
- [EmailJS](https://www.emailjs.com/)  
- [React Icons](https://react-icons.github.io/react-icons/)
